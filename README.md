# Timer & Stopwatch - Таймер и Секундомер

Приложение для отсчета времени с функциями таймера обратного отсчета и секундомера, разработанное на **Kotlin Multiplatform** с использованием **MVVM архитектуры**.

## 📋 Описание

Приложение предоставляет два основных экрана:

### 🕐 Таймер (Timer)
- Установка времени через TimePicker (часы, минуты, секунды)
- Обратный отсчет с точностью до секунды
- Большой дисплей отсчета (MM:SS)
- Круговой прогресс-индикатор
- Кнопки управления: Старт, Пауза, Сброс
- Уведомление при завершении таймера

### ⏱️ Секундомер (Stopwatch)
- Прямой отсчет с точностью до миллисекунды
- Дисплей времени (MM:SS.ms)
- Запись времени кругов (Lap Times)
- Кнопки управления: Старт, Пауза, Круг, Сброс
- Список всех записанных кругов с временем
